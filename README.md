# Rakshak.AI 🛡️
### AI-Driven Cyber Resilience Platform for Critical National Infrastructure (CNI)

Rakshak.AI is an autonomous cyber-defense platform designed to detect, analyze, and mitigate advanced threat campaigns targeting critical infrastructure (power grids, transportation, databases, etc.) in real-time.

---

## 🚀 Key Features

1. **Natural Language Attack Narrative:** Translates complex log files and alert sequences into a clear, chronological story of the attack.
2. **Graph Neural Network (GNN) Path Prediction:** Forecasts the attacker's most likely next steps in the network and preemptively hardens targets.
3. **Blast Radius Simulation:** Quantifies operational and financial risks before executing automated containment actions.
4. **Self-Learning Playbooks:** Analyzes response performance post-incident to autonomously suggest playbook optimizations.
5. **CERT-In Threat Intel Integration:** Correlates local anomalies with active threat advisories published by CERT-In and NCIIPC.
